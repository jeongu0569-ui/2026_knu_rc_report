# RC car 프로젝트 개발 로그

**개발 기간** : 2026.07.13 ~ 2026.07.17

---

# 월요일 (07.13)

## 오전

### 개발 내용
- 프로젝트 주제 선정 및 요구사항 분석
- 팀원 역할 분담
- 개발 환경 구축
  - Ubuntu 환경 확인
  - VS Code 설정
  - Python 및 uv 환경 구성
- 프로젝트 구조 설계

---

## 오후

### 개발 내용
- 팀 레포지토리 생성
- 각자 프로젝트 진행
    - 추후 유의미한 결과가 나온 코드를 짜집기하여 프로젝트를 진행할 예정

---

# 화요일 (07.14)

## 오전

### 개발 내용
- OpenCV를 이용한 차선(라인) 인식 알고리즘 구현
- 카메라 영상을 HSV 변환 및 이진화를 통해 차선 영역 추출
- 차선 중심 좌표를 계산하여 RC카의 조향값을 제어하는 자율주행 기능 구현
- 다양한 조명 환경에서 차선 인식 성능 테스트 및 파라미터 조정

### 결과
- 트랙의 차선을 인식하여 기본적인 자율주행 기능 구현 완료

---

## 오후

### 개발 내용
- LiDAR 센서를 이용한 장애물 인식 기능 구현
- 전방 일정 거리 이내 장애물 감지 알고리즘 적용
- 장애물 감지 시 차량 정지 및 회피 동작을 위한 제어 로직 설계
- 다양한 거리에서 장애물 인식 테스트 수행

### 결과
- 장애물 감지 및 차량 정지 기능 정상 동작 확인

---

# 수요일 (07.15)

## 오전

### 개발 내용
- 온디바이스 AI(TTS)를 활용한 음성 안내 기능 구현
- 장애물 감지 시 음성 경고 출력 기능 추가
- 자율주행 시작, 수동모드 전환 등 주요 이벤트에 대한 음성 안내 기능 설계
- Dashboard와 TTS 기능 연동 테스트

## 오후

### 개발 내용
- 자율주행, 장애물 인식, TTS 기능을 통합하여 시범 운행 실시
- 실제 트랙 환경에서 반복 주행 테스트 수행
- OpenCV 기반 차선 인식 과정에서 발생한 오인식 및 중심 좌표 계산 오류 분석

# 목요일 (07.16)

## 오전

### 개발 내용
- 주행 테스트 결과 RC카가 차선을 정상적으로 인식하여 추종은 수행
  - 영상 처리 및 제어 응답 속도의 지연으로 인해 차선을 즉각적으로 보정하지 못하는 현상이 발생
  - 차량이 차선을 중심으로 좌우로 반복 보정하는 S자 형태(Oscillation)의 주행 패턴을 보임
  - 차량의 속도를 줄여 주행 패턴 개선

- ppt 제작
