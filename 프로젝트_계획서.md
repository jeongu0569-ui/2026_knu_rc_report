# 5조 RC car 프로젝트 계획서

2026년 7월 13일

## 주제: RC Smart Drive - 온디바이스 AI 기반 자율주행 RC카 및 원격 제어 시스템

## 개인 역할 분담 및 실행

| 담당 | 역할 | 실행 내용 |
| --- | --- | --- |
| 홍기호 | PM / 자율주행 알고리즘 | 프로젝트 일정 관리, 요구사항 분석, 자율주행 알고리즘 설계, 발표 자료 제작 |
| 서정대 | RC카 제어 / 센서 | AutoCar API 연동, 모터 및 조향 제어, 카메라 및 센서 데이터 수집, 주행 테스트 |
| 한정우 | Backend / AI | Flask API 구현, 온디바이스 AI(TTS) 기능 구현, 자율주행 ↔ 수동주행 전환 로직 개발 |
| 김상엽 | Dashboard / UI | pywebview 기반 Dashboard 제작, 조이스틱 UI, 카메라 화면, 차량 상태 및 로그 시각화 |

## 과제 목표

- RC카가 지정된 트랙을 자율주행하도록 구현한다.
- 자율주행 중 사용자가 Dashboard의 조이스틱을 조작하면 즉시 수동 제어 모드로 전환되어 사용자의 조작을 수행한다.
- 수동 조작 이후 자율주행 버튼을 다시 누르면 현재 위치에서 자율주행을 재개하도록 구현한다.
- pywebview 기반 Dashboard를 제작하여 RC카를 제어하고 차량 상태를 실시간으로 확인할 수 있도록 한다.
- Dashboard에서 카메라 영상, 차량 상태, 주행 모드, 로그 정보를 실시간으로 표시한다.
- 온디바이스 AI(TTS)를 이용하여 주행 모드 변경 및 주요 이벤트를 음성으로 안내한다.
- Flask 기반 API를 이용하여 Dashboard와 RC카 간 명령 및 상태 정보를 주고받는다.
- 자율주행과 수동주행을 자연스럽게 전환하는 Hybrid Driving System을 구현한다.

## 수행 계획서

- AutoCar API를 이용하여 RC카의 모터 및 조향을 제어한다.
- 카메라를 이용하여 트랙을 인식하고 차선을 따라 자율주행하도록 구현한다.
- pywebview 기반 Dashboard를 제작하여 RC카 제어 화면을 구성한다.
- Dashboard에는 카메라 영상, 차량 상태, 속도, 조향각, 현재 주행 모드 등을 표시한다.
- Dashboard의 Virtual Joystick을 이용하여 수동 조작 기능을 구현한다.
- 사용자가 조이스틱을 움직이면 자율주행을 일시 중지하고 Manual Mode로 전환한다.
- Manual Mode에서 조이스틱 입력을 통해 RC카를 자유롭게 조작할 수 있도록 구현한다.
- Dashboard의 Auto 버튼을 누르면 현재 위치에서 다시 자율주행을 수행하도록 구현한다.
- Flask API를 이용하여 Dashboard와 RC카 간 제어 명령을 전달한다.
- 온디바이스 AI 기반 TTS를 적용하여 "자율주행 시작", "수동모드 전환", "자율주행 재개", "목적지 도착" 등의 음성 안내 기능을 구현한다.
- 차량의 주행 상태와 이벤트를 Dashboard의 Log 창에 기록하여 사용자에게 실시간으로 제공한다.
- 전체 시스템을 pywebview Desktop 환경에서 실행하여 최종 시연 환경을 구성한다.

## 시행 목적

본 프로젝트의 목적은 RC카를 이용하여 자율주행과 수동주행이 유기적으로 전환되는 Hybrid Driving System을 구현하는 것이다.

기존 자율주행 시스템은 사용자가 개입하기 어렵거나 수동 조작 이후 다시 자율주행을 시작하기 위해 프로그램을 재실행해야 하는 경우가 많다. 본 프로젝트에서는 사용자가 언제든지 Dashboard의 조이스틱을 이용하여 차량을 직접 제어할 수 있으며, 필요 시 다시 자율주행 모드로 복귀할 수 있도록 구현하여 실제 자율주행 차량의 운행 방식을 모사하고자 한다.

또한 pywebview 기반 Dashboard를 통해 차량의 현재 상태와 카메라 영상을 실시간으로 확인할 수 있으며, 온디바이스 AI 기반 TTS 기능을 적용하여 주행 모드 변경 및 주요 이벤트를 음성으로 안내함으로써 사용자 편의성을 향상시키는 것을 목표로 한다.

이를 통해 단순한 RC카 제어를 넘어 자율주행 알고리즘, 실시간 제어 시스템, Dashboard 개발, Flask 기반 API 통신, 온디바이스 AI 활용 기술을 하나의 시스템으로 통합하여 구현하는 경험을 얻고자 한다.

## 최종 시연 시나리오

1. pywebview Dashboard를 실행한다.
2. RC카가 트랙을 인식하여 자율주행을 시작한다.
3. Dashboard에서 실시간 카메라 영상과 차량 상태를 확인한다.
4. 사용자가 Virtual Joystick을 조작하면 차량이 즉시 Manual Mode로 전환되어 사용자의 입력에 따라 이동한다.
5. Dashboard의 Auto 버튼을 누르면 현재 위치에서 다시 자율주행을 수행한다.
6. 자율주행과 수동주행이 반복적으로 전환되는 모습을 시연한다.
7. 주행 모드 변경 시 TTS를 통해 "자율주행을 시작합니다.", "수동모드로 전환합니다.", "자율주행을 재개합니다." 등의 음성 안내가 출력된다.
8. Dashboard의 로그 창에서 주행 모드 변경, 차량 상태, 시스템 이벤트를 확인한다.
9. 최종적으로 Hybrid Driving System이 정상적으로 동작하는 과정을 설명하며 프로젝트를 마무리한다.

## 기대 효과

- RC카를 활용하여 자율주행과 원격 수동제어를 결합한 Hybrid Driving System을 구현할 수 있다.
- Flask API와 pywebview Dashboard를 이용한 실시간 제어 시스템 설계 및 구현 능력을 향상시킬 수 있다.
- Virtual Joystick과 AutoCar API를 연동하여 사용자 친화적인 RC카 제어 인터페이스를 구현할 수 있다.
- OpenCV 기반 영상 처리와 자율주행 알고리즘을 실제 RC카에 적용하는 경험을 얻을 수 있다.
- 온디바이스 AI(TTS)를 활용하여 사용자와 상호작용하는 스마트 제어 시스템을 구현할 수 있다.
- 실제 자율주행 차량에서 사용되는 자율주행 ↔ 운전자 개입 ↔ 자율주행 복귀 개념을 RC카 환경에서 구현함으로써 자율주행 - - - 시스템의 핵심 동작 원리를 이해할 수 있다.
- 향후 객체 인식, 장애물 회피, 음성 명령, 실시간 지도 생성 등의 기능을 추가하여 보다 고도화된 자율주행 플랫폼으로 확장할 수 있다.
